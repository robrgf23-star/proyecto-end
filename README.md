# proyecto-end


Aplicación de encuestas para usar en GitHub Pages. Usa localStorage para almacenar datos y permite exportar resultados.


## Cómo usar
1. Clona el repositorio.
2. Abre en VS Code o sube a GitHub.
3. Si usas GitHub Pages, publica la rama `main` y accede a la web.


## Archivos importantes
- `pages/` — formularios y reporte
- `js/` — lógica
- `css/` — estilos
- `data/` — referencia JSON
